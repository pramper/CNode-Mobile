# cnode-mobile

## 移动端CNode社区

使用Vue@1.0.26制作的移动端cnode社区，具有登录、发帖和评论等功能。

[线上地址](https://pramper.github.io/CNode-Mobile/dist)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
